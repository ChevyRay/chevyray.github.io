So You Want to Make Games?

Games are really fun to create, and these days there are tons of amazing tools you can pick up to make your own.

Before you decide you want to be a career game developer, I highly suggest picking up game development as a hobby. When you're a hobbyist, you can create whatever you want, and the games you make will help develop a nice portfolio.
