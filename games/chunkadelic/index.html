Chunkadelic

Chunkadelic is a psychadelic Flash game I made with Noel Berry (SkyTorn, Celeste).

You play a 3x3 grid of classic-style arcade games, with only one chance at each. When you finish one, you slide over to another, until all nine games are complete.

We created this game in only 48 hours for the Full Indie game jam.
