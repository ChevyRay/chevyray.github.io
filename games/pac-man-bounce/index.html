Pac-Man Bounce

I worked on this game as a contract programmer for Victory Square Games and BANDAI NAMCO.

This is a simple Pac-Man themed puzzle game where you have to arrange objects and redirect Pac-Man to finish each level. It features all the classic ghosts, with new behaviours designed to fit the game.

I coded all the game mechanics and the level editor the designers used to create the 100+ levels featured in the game.
